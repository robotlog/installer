# Siri Installer

### SiriUserbot Otomatik Deployer

[Siri UserBot Reposu](https://github.com/SiriUserBot/siriuserbot)

## Kurulum
```sh
git clone https://github.com/SiriUserBot/siriuserbot 
cd installer
pip install -r requirements.txt
python3 -m siri_installer
```
